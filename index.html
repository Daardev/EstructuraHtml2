Construye sitios ultra rápidos con Astro

Astro es un framework orientado a contenido que prioriza el rendimiento mediante renderizado estático por defecto y Islands Architecture.

¿Por qué Astro?

Rendimiento primero

Astro entrega HTML estático y carga JavaScript solo cuando es necesario. Resultado: mejores métricas como LCP, TTI y CLS.

Arquitectura de islas

Las “islas” son componentes interactivos aislados que hidratan de forma parcial. No toda la página necesita JS.

Integraciones y compatibilidad

Funciona con React, Vue, Svelte, Solid, Lit y más. Soporta Markdown/MDX, colecciones de contenido y adaptadores para múltiples hosts.

Cuándo usar Astro

  • Webs de marketing, blogs, documentación y e-commerce con foco SEO.
  • Proyectos donde el contenido es mayor que la interactividad.
  • Necesitas tiempos de carga y puntuaciones Lighthouse altos.

Inicio rápido

Paso básico de configuración local (ejemplo):

npm create astro@latest
cd mi-sitio-astro
npm install
npm run dev

Crea páginas en src/pages/ y componentes en src/components/.
Publica fácilmente en servicios como Vercel, Netlify o Cloudflare.
